# /ux

Evaluate user experience using the ux-reviewer agent.

## Usage
```
/ux [component or feature]
```

## Prompt

You are coordinating a UX review. Use the ux-reviewer agent to evaluate the user experience.

**UX Review Process:**
1. Identify the UI components or user flow to review
2. Launch the ux-reviewer agent
3. Check WCAG 2.1 AA accessibility compliance
4. Evaluate usability, consistency, and design patterns
5. Provide specific improvement recommendations

**Arguments provided:** $ARGUMENTS
